日陽起重工程團隊：北部、中部、南部起重，吊裝、搬運、安裝、定位、貨櫃裝卸

日陽起重公司是以從事半導體、TFT-LCD、太陽能及風力發電等綠能產業以及其他高科技產業等無塵室精密設備和重型機械設備搬運、安裝、定位起家的起重公司， 服務範圍除了桃園起重工程以外，足跡遍佈北部、中部、南部起重工程，並在各科技大廠及科學園區享有盛名。
公司成立的初始就直接跨入半導體產業起重工程：設備 Move In 的領域，日陽起重工程團隊成員，均是十幾年以上經驗的優質從業人員，憑藉著多年累積的經驗， 與不斷精進的設備搬運、重物搬運技術與起重機具，以及全公司職員定期參與勞工安全衛生講習，並堅守著日陽從業多年來 0 事故率的安全準則，來為客戶服務， 一直以來，均獲致客戶一致的信賴、肯定與好評。

『安全、效率、專業』為公司信念：日陽堅持『安全』為第一目標

安全：人員、設備的安全，不容打折。
效率：即時、順利、精準，是對客戶履行承諾(合約)的最佳方式。
專業：針對不同的作業項目(個案)使用不同的機具與最妥適的人員編組。

最嚴謹的工法、最專業的技術、最妥善的起重器具、最有效率的服務

我們透過購置新機具及設備，培育專業人才，以及工程現場的會勘，瞭解客戶的需求，並將工作環境中可能的危害因子納入思維，針對不同的作業項目， 擬定施工的方式，再與客戶討論，最後以適當的起重機具，並以嚴謹的工法，調配專業的技術人力，及備妥相關安全措施來為客戶提供最優質、最有效率的服務。
日陽不斷購入、更新各式起重器具，是為了使工程品質更高、更有效率，擁有的起重器具包括各式起重堆高機、手拉吊車、5mm 鋼板、各式油壓板車、活輪、戰車輪、吊裝平台、懸浮氣墊...等等。

日陽歡迎，短期承攬、長期承攬，以及起重相關工程承包

日陽起重有過豐富的短期承攬、長期承攬、工程承包經驗，承攬經驗包括：桃園華亞科學園區 - 美光科技半導體設備搬運、新北南林科技園區 - 南亞科技半導體設備搬運、 台灣日通北部 / 中部 / 南部起重工程承包、新竹科學園區 - 錸寶科技半導體設備搬運、台中太平洋氣門 - 3000 噸沖壓機定位及組裝、台南七股風力發電實驗區 - 風力發電設備組裝 / 拆卸...等等 從南到北的豐富、專業且安全的起重工程經驗；日陽也有多家在業界相當有口碑的合作夥伴，歡迎由上到下整套工程承包，從運輸，裝箱 / 拆箱、裝櫃 / 卸櫃、進廠 / 撤廠，到搬運、吊裝、定位...等等， 在全台(北部、中部、南部)都有密切配合的相關企業，若有任何需求，非常歡迎來信或來電與我們聯絡。
